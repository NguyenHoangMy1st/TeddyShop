# TeddyShop - Hệ thống mua bán gấu bông trực tuyến

## Mô tả dự án
TeddyBearShopping là một ứng dụng thương mại điện tử chuyên về bán gấu bông, được phát triển bằng Java Spring Boot. Ứng dụng cung cấp giao diện người dùng thân thiện và dễ sử dụng để mua sắm các sản phẩm gấu bông đa dạng.

## Công nghệ sử dụng
- **Backend**: 
  - Java 11+
  - Spring Boot 2.7.x
  - Spring Security
  - Spring Data JPA
  - Hibernate
  - Maven

- **Cơ sở dữ liệu**:
  - MySQL/PostgreSQL (tùy cấu hình)

- **Frontend**:
  - Thymeleaf
  - Bootstrap 5
  - JavaScript/jQuery

## Tính năng chính
- Đăng ký, đăng nhập và phân quyền người dùng
- Duyệt và tìm kiếm sản phẩm
- Giỏ hàng và thanh toán
- Quản lý đơn hàng
- Quản trị hệ thống (cho admin)

## Yêu cầu hệ thống
- Java JDK 11 hoặc mới hơn
- Maven 3.6+
- MySQL 8.0+ hoặc PostgreSQL 12+
- Git
